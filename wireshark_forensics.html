Filter Description
http Filter for HTTP traffic
ftp Filter for FTP traffic

telnet Filter for Telnet traffic
ssh Filter for SSH traffic
rdp Filter for RDP (Remote Desktop Protocol) traffic
icmp Filter for ICMP (Internet Control Message Protocol) traffic

ntp Filter for NTP (Network Time Protocol) traffic

nbns Filter for NetBIOS traffic
ssl Filter for SSL/TLS encrypted traffic

ip.addr == 192.168.1.1 Filter traffic to or from a specific IP address
ip.src == 192.168.1.1 Filter traffic from a specific IP address
ip.dst == 192.168.1.1 Filter traffic to a specific IP address
tcp.srcport == 80 Filter traffic from a specific port
tcp.dstport == 80 Filter traffic to a specific port
